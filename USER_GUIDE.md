# BPS Database Explorer - User Guide

## For Your Colleagues

This is a simple web-based tool to search and explore BPS (Biophysical Settings) models.

### How to Access

**If using Streamlit Cloud:**
- Just click the link provided (e.g., `https://bps-database-explorer.streamlit.app`)
- No installation needed!

**If running locally:**
- Double-click `setup_and_run.sh` (or run it in terminal)
- Wait for the browser to open automatically
- If it doesn't open, go to: `http://localhost:8501`

---

## How to Use

### 1. Search for Models

Type in the search box at the top. You can search by:
- **Model ID** (e.g., `10080`, `10080_1_2_3_7`)
- **BPS Name** (e.g., `Oak Woodland`, `Aspen Forest`)
- **Vegetation Type** (e.g., `Forest`, `Woodland`)
- **Keywords** (e.g., `oak`, `fire`, `mountain`)

### 2. View Results

Each result shows:
- **BPS Name** - The full name of the biophysical setting
- **Model ID** - Unique identifier for the model
- **Vegetation Type** - Type of vegetation
- **Map Zones** - Geographic zones where this model applies
- **Download Document** - Click to download the Word document

## Tips

- **Start broad**: Try searching for "Forest" or "Woodland" to see many results
- **Get specific**: Use exact Model IDs like "10080_1_2_3_7" for specific models
- **Use keywords**: Search for terms like "oak", "fire", "mountain" to find related models

---

## Troubleshooting

**Can't connect to the app?**
- Make sure the app is running (check the terminal/command prompt)
- Try refreshing the page
- Check the URL is correct

**Download button doesn't work?**
- The document file might not be available for that model
- Check your browser's download settings

**Need help?**
- Contact the person who set up the app
- Check the SHARING_GUIDE.md for technical details
